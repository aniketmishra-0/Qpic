How to Use Qpic
Watch the walkthrough or follow the steps below

Overview

1
App bar Switch between Crop and Rename Batch tools. Use the theme switcher (light / dark / system) on the right.
2
Left panel — What to Crop Toggle Questions / Solutions, set page ranges, enable Smart mode and Online AI mode.
3
Right panel — Output Options Choose format (PNG/JPEG), DPI, file prefix, and start number for the exported images.
4
Light & Dark themes The app follows your OS preference by default. Override it anytime with the theme switcher in the top-right corner.

Cropping Questions

1
Upload your PDF Click "Choose PDF" or drag-and-drop your MCQ question paper onto the upload area.
2
Set options Choose question numbering style, toggle Smart mode (auto-detect layout), and enable Online AI mode if you have an API key.
3
Analyze & Review Click "Analyze & Review". Detected question boxes appear over the page. Fix any wrong boxes by clicking Fix → drag the correct region.
4
Finalize & Download Click "Finalize & Download" to get a ZIP of cropped images. Choose Combined, Questions only, or Solutions only.

Batch Renaming Images

1
Switch to Rename Batch tab Click "Rename Batch" in the top app bar to open the rename tool.
2
Upload images Drag-and-drop or select multiple image files (PNG, JPEG, etc.) you want to rename.
3
Set a naming pattern Type a prefix (e.g. "Q") and set the start number. Use the Variables button to insert dynamic tokens like {n} for auto-numbering.
4
Rename & Download ZIP Preview the new names in the gallery, remove any unwanted files, then click "Rename & Download ZIP".
